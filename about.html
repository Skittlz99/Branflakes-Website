The Process

How It Was Built.

The goal of this project was simple: learn how AI-assisted development actually works in practice. Specifically, using the Claude Code extension inside Visual Studio Code to vibe code an entire website from scratch — brand identity, copy, layout, and deployment — without opening a single traditional design tool.

The result is Branflakes. Every line of HTML, every animation, every card and color token was generated through conversation. What follows is exactly how it was done.

01

Install the Claude Code Extension in VS Code

Added the Claude Code extension directly inside Visual Studio Code. This brings the AI into the same environment where the code lives — no context switching, no copy-pasting between windows. The editor and the assistant become one interface.

02

Create Markdown Skill & Config Files

Created a set of .md files to give Claude structured, persistent instructions. A CLAUDE.md held the brand rules, design constraints, and hard guardrails. Separate skill files defined specific AI behaviors — for example, a frontend-design skill enforcing high-craft UI standards, a commit skill for consistent git messages, and a simplify skill for reviewing and cleaning generated code.

03

Write Detailed Prompts

Wrote precise, intentional prompts for every section of the site — specifying the layout, the audience, the tone, the sections, the copy direction, and the visual treatment. The quality of the output is directly tied to the specificity of the input. Vague prompts produce vague results. A detailed brief produces production-ready code.

04

GitHub Repository & Vercel Deployment

Once the site was built, initialized a GitHub repository and pushed the project directly from VS Code. Connected the repo to Vercel for automatic deployment — taking the site from local files to a live public URL in under five minutes. Every subsequent push triggers a new deploy automatically.

Built with Claude Code in VS Code

Claude Code is Anthropic's official AI coding assistant, available as a VS Code extension. This entire project — architecture, implementation, copy, iteration, and git workflow — was executed through a single conversational session inside the editor. No IDE switching. No context loss. One interface, end to end.
