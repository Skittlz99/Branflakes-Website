The Origin

It started with a nickname.

In middle school, Branden had a nickname: Branflakes. It wasn't subtle. It wasn't particularly cool. It was, at best, a dietary pun lobbed across a cafeteria and somehow made to stick — the way all the best nicknames do.

It stuck for years. Then it got shelved. Then, roughly a decade and several lines of production code later, it got resurrected — not as a nostalgic joke, but as something else entirely: a conceptual brand. A design exercise. A statement about what happens when you apply serious craft to a ridiculous premise.

Branflakes is that premise executed without apology. A high-end SaaS brand built around breakfast cereal. Enterprise-grade aesthetics applied to a bowl. Because if you're going to do something, do it like you mean it.

The founder — Branden — is a developer and designer who spends too much time thinking about design systems, and apparently not enough time thinking about whether his middle school nickname should have stayed buried.

The Process

How It Was Built.

This entire project is an exercise in vibe coding — the emerging practice of building production-quality software through structured, intent-driven collaboration with AI.

No templates were purchased. No Figma files were dragged off a marketplace. The design system, the brand guidelines, the color tokens, the typographic hierarchy — all of it was defined through precise, high-context prompting, then executed in code by Claude Code. Every pixel of layout, every hover state, every staggered animation was specified in language and rendered in markup.

The brand visuals — the logo, the cereal box concept — were generated using custom AI image generation prompts, then integrated into the design system as real assets. Not placeholders. Not lorem ipsum equivalents. Real brand artifacts, generated from scratch.

The goal was simple: practice the full stack of modern, AI-assisted web development. Prove that with the right constraints, the right prompts, and a clear design vision, you can build something that looks and feels like it came out of a San Francisco agency — because the craft was always the point.

01

Design System Prompting

Color tokens, typographic rules, spacing systems, and motion specs written into a CLAUDE.md as hard constraints. AI reads them as law.

02

AI Brand Asset Generation

Logo mark and cereal box concept generated via custom image AI. Iterated on geometric data-bowl motifs until the mark felt ownable.

03

Vibe-Coded with Claude Code

Full frontend — markup, animations, layout systems — generated via Claude Code using structured, high-context prompts. Tailwind CDN as a utility layer over custom tokens.

04

Screenshot, Compare, Iterate

Automated screenshot loops using Puppeteer. Each output compared against design intent in real-time. Design corrections made in natural language.

Built with Claude Code

Claude Code is Anthropic's official AI coding assistant. This project used it as the primary development environment: planning, scaffolding, writing, and debugging all happened through conversation. Zero context switching. One interface, end to end.
