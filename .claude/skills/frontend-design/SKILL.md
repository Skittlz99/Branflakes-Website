# Frontend Design Skill
When active, prioritize these standards:
- **No AI Slop:** Never use default Tailwind colors (blue/indigo) or fonts.
- **High-End UI:** Use custom brand palettes, layered shadows, and intentional typography pairings.
- **Mobile First:** Follow all touch-target and spacing rules from CLAUDE.md.
